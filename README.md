# Primeiro de tudo deixa eu explicar como surgiu esse projeto.

Esse projeto surgiu de uma necessidade de analisar e descobrir a quantidade de acessos em um site, aos poucos foram surgindo ideias e sendo testadas, porque um portofólio de fotos? na epóca tirava muitas fotos e queria divulgar, foi o começo para melhorar a função de compressão de imagem, comecei a tentar melhorar a visualização mobile então desenvolvi uma analise mais aprofundada de quais dospositivos e suas caracteristicas, queria saber qual era a foto mais acessada usar um analisador de click, ao desenvolver ele tava salvando tudo, até meus acesso de desenvolvimento então criei um função para eliminar a coleta de dados.

## Visão Geral do Projeto

Este é um projeto em [Firebase](https://firebase.google.com/)

Tecnologias e Frameworks usados são

* [Authentication](https://firebase.google.com/docs/auth/web/start) para permitir que os usuários façam login no seu app.
* [Cloud Firestore](https://firebase.google.com/docs/firestore/quickstart) para adicionar dados e ver os dados adicionados.
* [Cloud Storage](https://firebase.google.com/docs/storage/web/start) para armazenamento de dados com alta disponibilidade.
* [Bootstrap](https://getbootstrap.com.br) para projetos responsivos para dispositivos móveis na web.
* [SB Admin](https://startbootstrap.com/templates/sb-admin/) para o painel em Bootstrap 4 com um estilo personalizado.
